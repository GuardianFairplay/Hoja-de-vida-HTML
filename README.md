# Asistente Virtual – Nicolás Uribe

Este proyecto presenta un asistente virtual creado para facilitar tareas cotidianas, responder preguntas y automatizar acciones simples. Está diseñado para ser claro, ágil y fácil de usar.

## Características
- Respuestas rápidas a consultas comunes  
- Gestión básica de recordatorios  
- Ayuda en organización personal  
- Interfaz sencilla pensada para cualquier usuario  

## Objetivo
Ofrecer una herramienta que apoye el trabajo diario reduciendo tareas repetitivas y proporcionando información de forma inmediata.

## Tecnologías
- JavaScript  
- HTML / CSS  
- Node.js (dependiendo de la versión implementada)

## Uso
1. Clona el repositorio.  
2. Instala dependencias si es necesario.  
3. Ejecuta el proyecto o abre el archivo principal en el navegador.
